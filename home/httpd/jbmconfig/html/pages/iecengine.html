Welcome to the Straton Remote Link Walkthrough

Straton Remote Link will allow downloading and extracting straton databases through this device's cellular connection. Any straton database access supported from the straton link can be supported remotely through this feature.


Next

Security

Since this remote unit could be reconfigured or reloaded remotely, we may want to enable some protection against this access. Once the Straton Remote Link is activated, it will only stay open for a few minutes. New attempts after this timeout will be rejected, but current connections will not be prematurely terminated.

Access can also be restricted to a specific IP address that you provide. If you will be accessing the straton remote link from the same PC that you are using right now, we can detect the IP address you are using now and provide that as the only allowed IP address.
