Description

Persistent mode, the unit will always attempt to maintain a constant connection to the POTS network. In On-Demand mode, the connection to the POTS network will only be attempted when packets are destined to leave the modem's PPP interface. In addition, after a period of Idle Time, the connection will terminate.

On-Demand, the unit is commonly chosen for an environment using an ethernet connection as a primary interface, while IP Fallback is used to bring up this dialup connection as a backup.

Recommended Setting

Persistent - When this link is intended as a Primary network connection.
On-Demand - When this link is intended as a Fallback network connection.
