Description

Choose the Action Type for the event.

None: No action, log the event only
Send SMS Message: Send an SMS message to a single recipient. Use multiple Events to notify more than one contact.
Send Email Message: Send an Email message to a single or multiple recipient(s).
Write IODB Value: Write to a known IODB register. Be careful not to assign the write register to be the same as the event source or it will get a config error.
Run Command Script: Run a Command Script that performs an Action.
SVM Alert Message: Send an alert message to the SVM server that appears in unit history.
