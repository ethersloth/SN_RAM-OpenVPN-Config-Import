Description

Input Type:
Select the type of signal that will be connected to the input.

The Disabled option will completely disable the channel so the channel will always report a zero. The Input Range configuration will be disabled as well.
When Voltage Range is selected the analog input will be configured to take a DC voltage range. The Input Range must also be configured.
With Current Range is selected the analog input will be configured to take current instrumentation input (0-40mA or 4-20mA). The Input Range must also be configured.
