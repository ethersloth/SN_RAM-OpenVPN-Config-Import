Description

The events log contains the status of each event configured on your device and each line consists of 7 fields, comma separated as described below.

Each line of events log format include:

  • Date/Time
  • Event Number
  • Event Name ("N/A" if no name)
  • Event Condition/Status (1/0)
  • Event Condition/Status (Active/Inactive)
  • Event Data Source Value (0 at initial time)
  • Description (optional)
