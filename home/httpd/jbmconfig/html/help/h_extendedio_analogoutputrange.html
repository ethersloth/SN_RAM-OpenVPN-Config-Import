Description

Output Range:
Select the type of signal to be supplied by the output channels.

The Disabled option will completely disable the channel.
When 0-5VDC is selected the analog output will be configured to supply a DC voltage from 0-5VDC. The output voltage will be scaled to 0-32767 (decimal value in the Modbus Address field).
When 4-20mA is selected the analog output will be configured to supply a current from 4-20mA. The output voltage will be scaled to 0-32767 (decimal value in the Modbus Address field).
