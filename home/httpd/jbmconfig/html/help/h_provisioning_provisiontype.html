Description

Select the type of provisioning to be performed by selecting from one of the following options:

  • Vision IP Only:
    • - Standard Sprint Provisioning.
  • Static IP Only:
    • - Must be chosen to receive a Sprint Static IP.
  • Static, Vision Alternate
    • - Preferred behavior will be to use a Sprint Static IP.
    When unavailable, we will switch to Vision for a specified time frame until trying Static mode again.
  • Datalink Only:
    • - Sprint Datalink requires many additional fields.
    Only select this if your dataplan supports Datalink, and you have been provided these settings.
  • Datalink, Vision Alternate:
    • - Preferred behavior will be to use a Sprint Datalink profile.
    When unavailable, we will switch to Vision for a specified time frame until trying Datalink mode again.
