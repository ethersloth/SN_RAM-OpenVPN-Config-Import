Description

Enter a valid port number (1-65535) to be used for the connection. (Required)

Recommended Setting

Take care to choose a port number not already used by other system services. Consult Status->Network->Socket Statuses->TCP Only for a list of ports currently in use. Please note that a Firewall Allow rule will need to be added for remote access via
Network->Firewall->Port Allow/Forwarding Rules->Service Access Rules.
