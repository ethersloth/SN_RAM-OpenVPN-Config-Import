Description

Select the Mobile IP Mode by choosing one of the options available in he provided drop-down list. The following choices are available:

  • Mobile IP Preferred
    • - Attempt to connect using MIP, but allow a fallback to Simple IP (SIP) when MIP is unavailable.
  • Mobile IP Only
    • - Will not allow SIP connections and will continue to retry MIP conections only.

Recommended Setting

Mobile IP Preferred.
