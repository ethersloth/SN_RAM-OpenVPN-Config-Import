Description

Bridge Mode: If 'Enable' is selected, the Wi-Fi interface will be bridged with the eth1 interface, allowing both interfaces to be configured as one single device. This will allow all traffic on eth1 to pass to the Wi-Fi interface and vice versa. If 'Disabled', the Wi-Fi and ethernet interfaces will be separated for configuration and clients on one interface will not see traffic on the other interface. Most users should choose 'Enable' here.
