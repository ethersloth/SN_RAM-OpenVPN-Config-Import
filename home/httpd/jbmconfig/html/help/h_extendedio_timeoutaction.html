Description

Drop Physical Outputs:
This option is tied to the "CPU Monitor Timeout" field. When a timeout to the CPU occurs the selected "Timeout Action" will take effect. When "Drop Physical Outputs" is selected, all outputs are dropped to an OFF state. In the "No Action" mode outputs will hold their last known value. In a discrete output module, the OFF state is simply turning the outputs off, in an Analog Output module, OFF means to set all outputs to a nominal calibrated zero output. TPO Outputs will be set to output 0, but may require the normal TPO interval to elapse before the output will go off.
