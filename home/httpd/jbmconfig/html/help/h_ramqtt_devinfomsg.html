Description

Note: GPS Latitude and Longitude are included in a separate section of the Basic and Full Device Info Messages. While the Latitude and Longitude values are included in the same message, they might not be displayed on the cloud side visualization in the same area as the other device info values.

Choose the type of Device Info message to be sent. This is in addition to the I/O data point update messages.

None - Sends no message.
Basic - Sends a basic/small, predefined list of device information values to the cloud.

  • Unit name
  • Serial Number
  • Cell IP
  • Up Time
  • Version
  • Model
Full - Sends a full, predefined list of device information values to the cloud.
  • Version long (ex: 4.24.99.0)
  • Host Name
  • Unit Temp
  • Cell modem firmware version
  • PWR1 Volt
  • Cell Module Temp
  • All active interface IPs
  • Cell Servcice Type
  • MDN
  • Cell Carrier PLMN
  • Cell Current Channel
  • ECIO
  • RSRP
  • RSRQ
  • Cell PRL Version
  • Sim ID / ICCID
  • Sim IMSI
  • Wireless UpTime
  • IMEI
  • Wireless Signal
  • RSSI
  • Current APN
  • GPS Altitude
  • GPS Time
  • GPS Number of Satellites
  • GPS Feet From Centerpoint
  • GPS Speed
