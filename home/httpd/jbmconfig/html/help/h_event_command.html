Description

Choose the name of the command script to be executed when the Event is True.

None: Standard operation with no special behaviors.
Rotate Data Logs: Rotate running data logs.
Create Data Log Entry: Create new data log entry.
Restart IPSec: Restart the IPSec service. ie: Bring the IPSec tunnel down, then reestablish the tunnel.
Stop IPSec: Stop the IPSec service and do not reestablish the tunnel
Restart Modbus: Restart modbus services. Reboot: Reboot the entire device.
Restart Serial IP: Restart the Serial IP service. Cellular Reset: Restart the Cellular Module.
Cellular Data Temporary Off: Disable Cellular Data connection temporarily.
Cellular Data Temporary On: Enable Cellular Data connection temporarily.
Cellular Power Temporary Off: Power off the Cellular Module temporarily.
Cellular Power Temporary On: Power on the Cellular Module, does not update config.
