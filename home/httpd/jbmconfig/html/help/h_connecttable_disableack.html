Description

Transparent Mode: Unused.

Visa Mode: Once a message is received from the serial device (ATM/POS) and the LRC is valid, this will disable sending an ACK. Certain ATM/POS devices will fail if sent an ACK, and rather used the response message from the TCP host as an implied ACK. Certain ATM/POS devices require an ACK before receiving the response message from the TCP host.

Recommended Setting

No
