Description

Enables device to send Cell modem RF status information at very short intervals.

This feature is only supported by SixView Manager Servers running 1.22 or later version server software.

Note: Each Rapid RF check-in will produce about 1300 bytes of data. Setting this value to 15 seconds will generate about 7.4MB of data a day (224.6MB every 30 days). This amount is addition to any other data the unit might be transmitting/receiving.

Recommended Setting

0 if you are not using this feature on a SixView Manage server.
