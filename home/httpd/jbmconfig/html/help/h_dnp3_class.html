Description

Class Assignments: This field is activated on both Levels 2 and 2+. It's used to determine if a DNP point will generate events. In case a DNP point generates events (Object2 Binary Change Events) then it should be associated to a class (Class 1, Class2 or Class 3), otherwise it should be associated to None. By default all DNP Points don't generate events, this feature should be modified by the user.
