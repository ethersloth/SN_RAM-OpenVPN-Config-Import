Description

Enable the Crimson SMS API interface on port 1000. See Crimson HOWTO guide for more information and instructions on how to configure your Crimson application to connect to these SMS services.

SMS API Interface Options
No: SMS API Disabled
TCP API Only: Allows you to send SMS messages via TCP
TCP and Serial API: Allows you to send SMS messages via TCP and Serial Port

NOTE: If using the "and Serial" option, it will prevent other serial port activity such as Modbus. Choose "TCP Only" if you are also using protocol conversion on the serial port.
