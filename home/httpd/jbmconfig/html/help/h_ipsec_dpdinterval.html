Description

Set the delay (in seconds) between Dead Peer Dectection keepalives that are sent for this connection.

A low value will rapidly discover a broken tunnel, however a lot of bandwidth will be used and this is not ideal for a limited data plan (i.e. cellular telemetry plan).

(Required)

Recommended Setting

Unlimited Data Plans: 30-60 sec.
Limited Data Plans: 1800 sec. (30 minutes)
