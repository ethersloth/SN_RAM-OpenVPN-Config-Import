Description

Enter the address of internal Modbus address for Digital Output I/O control.

Valid Address

Zero Based Addressing: 0 through 65535 as defined for specified I/O type.
One Based Addressing: 1 through 65536 as defined for specified I/O type.

NOTE: The address ranges are displayed on I/O Transfer screen under Display Of Modbus Default Slave Addresses based on configured local register allocation for specified I/O type.
