Description

Set the local bind IP address for tunneled packets. This is useful when the tunnel should not originate on the default interface. Examples include a GRE tunnel used on and internal LAN interface, or inside another tunnel such as IPSec. If your tunnel is not working, try entering the IP of the source interface here. (Optional)

Note: If supplied, the Local IP Address must be an address on another interface of this host. If not supplied, tunneled packets can be received from any interface.
