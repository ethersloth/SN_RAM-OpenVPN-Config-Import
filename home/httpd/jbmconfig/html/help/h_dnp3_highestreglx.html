Description

Highest Register Address (LX512 - LX1023): The mapping starts at the DNP Analog Input Point 9216 for LX512 up to the DNP Analog Input point whose address is the configured LX Highest Register Address - 512 + 9216 (max. 9727 for LX1023).

This field is used to show or set the highest register address to map DNP points. If Configure DNP Points option is set to Yes, the Highest Register Address field is displayed to enter a Highest Register Address value.
