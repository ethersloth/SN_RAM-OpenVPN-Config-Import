Description

The Time Window is the amount of time the data is being scanned for changes in rate in seconds. The time window is a sliding window, meaning that as time goes by, the time window slides along.

For example if we set time window to 10 minutes (tw = 600), this means that LAST 10 minutes of data will be used for calculation.

Minimum value
Minimum value for time window is 2, which means calculations will be done on current data value and one data value from last sample which was two seconds ago.
