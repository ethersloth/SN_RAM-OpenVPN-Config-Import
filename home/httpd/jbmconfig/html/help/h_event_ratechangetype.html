Description

Choose which data rate of change type to use for this event.
Min-Max: Takes the lowest point and the highest point within given time window and calculates the difference between the two. If the difference between minimum point and maximum point is grater than Change Limit, then an alarm will be activated.

Event is activated when (Max - Min) > Change Limit and clears when (Max - Min) <= Change Limit within given time window.

Average: Takes all the data rate points within the current time window and calculates the average rate. Then average rate is compared to change limit. If Average rate is grater then change limit, then the alarm is activated.

Event is activated when Average > Change Limit and clears when Average <= Change Limit within given time window.

For example:

Avg = SUM((|dv/t|)1 + (|dv/t|)2 + ... + (|dv/t|)N) / N.

Where Avg = average, dv = data value, t = time and N = number of samples in time window.
