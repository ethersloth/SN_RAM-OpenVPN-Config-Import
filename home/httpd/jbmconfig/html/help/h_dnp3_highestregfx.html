Description

Highest Register Address (FX0 - FX1023): The mapping starts at the DNP Analog Input Point 8192 for FX0 up to the DNP Analog Input point whose address is the configured FX Highest Register Address + 8192 (max. 9215 for FX1023).

This field is used to show or set the highest register address to map DNP points. If Configure DNP Points option is set to Yes, the Highest Register Address field is displayed to enter a Highest Register Address value.
