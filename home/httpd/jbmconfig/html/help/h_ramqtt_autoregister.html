Description

This option determines whether RAMQTT will attempt to create a device instance on the Cloud and obtain generated credentials or use existing credentials for an already existing device instance.

Auto Registration - RAMQTT will attempt to create a device instance on the Cloud and obtain generated credentials for the newly created device. If the device already exists, the creation of a new device will be skipped and the credentials for the existing device will be used.

Already Registered - RAMQTT will not attempt to automatically create the device instance on the Cloud or obtain generated credentials. It will instead connect using the credentials provided. This option assumes that the device instance on the Cloud already exists.
