Description

To allow ICMP echo responses (Ping) from external devices through untrusted interfaces on this unit, select Yes, otherwise select No.

To restrict access via a configured Whitelist, select a Whitelist name from the list of names available in the drop-down list box provided.

Whitelists may be viewed/defined via the Network->Firewall->ACL Rules->Subnet Whitelist Rules screen.

Note: This setting will NOT overide any firewall rules defined on other pages, such as service access (allow) or redirect rules.

Recommended Setting

Yes
