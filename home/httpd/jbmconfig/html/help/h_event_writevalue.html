Description

Choose what to write into the IODB register.

Data Source: will write the input (data source) of the event. (Available if data source and write types match.)
Event Condition: will write a 1 = True or 0 = False based on this event's condition/status.
Fixed Value: will write a constant fixed number to that entry.
Counter: will increment the value in the IODB location by one. (Available if write type is not discrete.)
