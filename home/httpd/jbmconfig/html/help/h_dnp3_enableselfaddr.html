Description

Enable Self Address: The DNP3 Slave Driver can be configured to send its own DNP Address when a DNP3 Master asks for it. When this option is set to Yes, if a message is sent with the Self Address (65532) in the destination address field, it will respond with its unique individual address. This feature simplifies the commissioning, troubleshooting and maintenance of devices with an unknown address. If this feature is not enabled, the station will ignore the messages sent to the Self Address.
