Description

Count Value Size:
This is a plain counter mode that is either 16 or 32-bits in size, which counts on positive edge or negative edge depending on the polarity bit. When the 16-bit Count Returned option is selected, the count will increment in a single register from 0 to 65535, then roll over to 1 and continue to count upward again. Select 32-bit Count Returned to use two consecutive 16-bit registers as one 32-bit register. When the first register rolls over to 1, the second register begins counting at 65536 (second register is most significant) and continue counting upward in 32 bit mode.
Note: Two consecutive registers are always allocated in the "CNT Modbus Address" column whether set for 16-bit or 32-bit mode. Therefore, when using "16-bit Count Returned" mode the second register should be ignored.
