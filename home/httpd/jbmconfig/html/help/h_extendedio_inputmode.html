Description

Input Mode:
The "Input Mode" defines filtering mode of the Discrete input channel. Choosing Disabled will completely disable the channel so that will report a zero. When Slow Response (filtered) is selected, the discrete input will have software filtering applied to the input. Software filtering is especially suitable when the input is connected to a mechanical switch or relay because it will eliminate contact bounce. In this mode counting is limited to a maximum of 10Hz.
When Fast Response (no filtering) is selected, the discrete input will have no filtering applied to the input. This especially suitable for solid state switches where no contact bounce is present.
