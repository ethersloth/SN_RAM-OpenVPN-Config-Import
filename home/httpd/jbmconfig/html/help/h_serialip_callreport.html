Description

If this option is selected as Yes to log the call event, you need to specify a file name in Call Reporting File Name field. This option force the unit to log all the event for serial communication. Bellow is the list of all supported events.

Events: Julian Date, Time Stamp, Port ID, Dial, Answer, Modem Connect Speed, Number of Bytes Received, Number of Bytes Transmit, ENQ's Indicator, Bad LRC Indicator, Retransmit Indicator, Number of Transactions, Number of Bytes That Host Received, Number of Bytes That Host Transmit, Host Processing Time in Seconds, Total Time From Start to Finish, etc.

Recommended Setting

Yes, if call events is desired.
