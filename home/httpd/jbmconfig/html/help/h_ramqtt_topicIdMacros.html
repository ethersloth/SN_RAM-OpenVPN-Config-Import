Description

The Base Topic is the user defined topic that all RAMQTT's endpoint topics will be appended to.
<BaseTopic>/iodb
<BaseTopic>/deviceInfo
<BaseTopic>/cmd
<BaseTopic>/cmdResponse
<BaseTopic>/lwt

Base Topics support topic macros that can reference unique information about the device within the topic itself.
%c - clientId
%m - mac address of eth0 (alphanumeric, no special characters)
%i - imei
%s - serial number
%j - iccid
%p - ip address of eth0

Examples:
test/topic/%s
test/topic/971X12345678912

company/%i/%c
company/123456789123456/test_client
