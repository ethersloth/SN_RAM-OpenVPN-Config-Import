Description

Highest Register Address (FY0 - FY1023): The mapping starts at the DNP Analog Output Point 8192 for FY0 up to the DNP Analog Output point whose address is the configured FY Highest Register Address + 8192 (max. 9215 for FY1023).

This field is used to show or set the highest register address to map DNP points. If Configure DNP Points option is set to Yes, the Highest Register Address field is displayed to enter a Highest Register Address value.
