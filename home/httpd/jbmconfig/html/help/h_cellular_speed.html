Description

Select the connection speed to be used for the cellular modem connection from the drop-down list provided. Auto will use the widest available defaults, starting at the highest speed available (4G-LTE, if present).
Possible values include:

  • Default
    • - Do not adjust the module settings.
  • Auto
  • LTE
    • - 4G/LTE only
  • 2G3G
    • - 3G/2G only, no LTE
  • 3G Only
    • - 3G only service
  • 2G Only
    • - 2G only service

Recommended Setting

Auto

or

Default
