Description

Enter the number of minutes the SixView Manager Client process should wait before connecting to the SixView Manager server. (Required)

Note: While lower values can result in more timely status reports with the SixView Manager Server, it comes at an expense of increased data traffic, which may be an issue when the connection utilizes a cellular modem with a sevice plan where cost is based on bandwidth usage.

Recommended Setting

A value of 220 is suggested for Cellular carriers that use an inactivity timeout of four hours.
