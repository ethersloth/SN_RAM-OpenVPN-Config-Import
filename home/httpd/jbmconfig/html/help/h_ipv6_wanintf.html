Description

Specify the IPv6 Upsteam Router path. If a unit has access to a real IPv6 router on multiple interfaces, you may specify it here. Cellular devices expect that the wwan0 interface will lead to the IPv6 routers. Wired Routers will expect that eth0 (default untrusted/external interface) may also lead to an upstream IPv6 router.

A reboot is required after changes to IPv6 Routing configuration.

Recommended: Auto.
