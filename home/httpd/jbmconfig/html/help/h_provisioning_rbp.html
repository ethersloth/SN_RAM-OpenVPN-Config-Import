Description

Select the Roaming/Band preference by choosing one of the options available in he provided drop-down list. The following choices are available:

  • Default
  • Automatic
  • Home Only
  • A-Band Only
  • B-Band Only

This option determines whether or not the cellular modem is allowed to enter roaming mode and, when roaming is enabled, the preferred roaming band (or an automatic selection). This option may not be supported by all cellular modem card/modules, and not all cards may have dual band capability.

Default mode will not modify the cellular car/module's current settings.
Automatic mode will attempt to choose the best option for the current carrier, which includes allowing roaming.

Recommended Setting

Default.
