Description

Enable Real Time Data Traces: The DNP3 Slave Driver can be configured to generate real time traces of every Master-Slave DNP3 transaction for diagnosis and debugging purposes. The real time communication data traces can be enabled/disabled at any time. The output will be directed to a text file within the file system for later upload. When your trace scenario is complete, simply run a gatherstats to automatically include this DNP3 trace. Then contact Support with your results.
