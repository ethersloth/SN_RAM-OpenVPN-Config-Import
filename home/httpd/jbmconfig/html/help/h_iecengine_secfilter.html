Description

Connections from the IP (or IP range) listed here will be allowed to connect to the Straton Remote Link enabled device.

None - Allow World: This will allow connections from any remote IP Address. Choose this if you are unsure what IP to enter, or if multiple people may access this link.

Allow Currently Connected IP Only: This will autodetect the IP in use from this current browser session. If you are running PC Workbench (RITE) on the same PC that is accessing this page in a browser, try this option first.

Allow Specific IP: Enter a specific IP in the Filter From field. Only this IP will be allowed to connect to the Straton Remote Link. If your endpoints are connecting through a firewall, a computer's assigned IP might not be the same IP used when connecting to this remote unit.
