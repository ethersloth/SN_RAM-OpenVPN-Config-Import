Description

Select the Access Security profile for who will be allowed access to the device.

Available Options

Allow any number: Any number can access features for all user types. Login is still required.

Allow any number, Admin must be on whitelist: Any number can access Basic/Tech functions, but only Whitelist users can access Admin functions. Login is still required.

Only allow Whitelist numbers: Only Whitelist users can login. All users will need to login to have the functions of their access level, non-whitelist users' requests will be ignored.
