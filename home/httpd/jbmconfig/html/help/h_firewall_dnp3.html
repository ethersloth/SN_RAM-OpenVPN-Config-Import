Description

To allow external devices to connect to the local DNP3 Server through untrusted interfaces on this unit, select Yes, otherwise select No. This defaults to port 20000, but is controlled by the listening port chosen in the Automation->DNP3->Physical Link Layer screen.

To restrict access via a configured Whitelist, select a Whitelist name from the list of names available in the drop-down list box provided.

Whitelists may be viewed/defined via the Network->Firewall->ACL Rules->Subnet Whitelist Rules screen.

Note:

Setting this option to Yes does not enable the DNP3 Server, it just allows it to be accessible via the firewall when it is enabled. The DNP3 Server may be enabled via the Automation->DNP3->Physical Link Layer screen.
