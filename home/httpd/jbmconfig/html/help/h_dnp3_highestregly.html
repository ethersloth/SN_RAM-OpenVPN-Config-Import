Description

Highest Register Address (LY0 - LY1023): The mapping starts at the DNP Analog Input Point 9216 for LY0 up to the DNP Analog Input point whose address is the configured LY Highest Register Address + 9216 (max. 10239 for LY1023).

This field is used to show or set the highest register address to map DNP points. If Configure DNP Points option is set to Yes, the Highest Register Address field is displayed to enter a Highest Register Address value.
