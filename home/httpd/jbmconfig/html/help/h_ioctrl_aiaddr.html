Description

Enter the address of internal Modbus address for Analog Input I/O control.

Valid Address

Zero Based Addressing: 0 through value of defined registers configured for specified I/O type minus 1.
One Based Addressing: 1 through value of defined registers configured for specified I/O type.

NOTE: The address ranges are displayed on I/O Transfer screen under Display Of Modbus Default Slave Addresses based on configured local register allocation for specified I/O type.
