Description

Select Yes to allow routing through the IP Transparency device behind the selected interface. This means that the ISP has setup routing to/from a subnet or subnets which will be located behind the IP Transparency device located behind this device.

Recommended Setting

Select No for common operation

WARNING: If your ISP for your Cellmodem PPP connection is NOT allowing routing, then this option must be turned off. Enabling this option without consulting your Modem Carrier and having routing enabled on your account can cause the PPP connection to DROP every time the modem makes a connection. This should only be enabled for SPRINT LAC/LNS at this time.
