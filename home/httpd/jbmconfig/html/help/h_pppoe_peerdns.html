Description

Select the method by which DNS Server information should be obtained.

Choices include:

  • Use Unit Default
    • - Do not obtain DNS information from PPPoE Server. Use settings from Network→DNS Settings instead.
  • Use Peer DNS
    • - DNS information should be obtained from the peer host once connected.
  • Use Custom DNS
    • - DNS information is entered manually in the fields which will appear below.

Recommended Setting

Use Peer DNS
