Alarm Severity
The Alarm Severity determines how severe the event is when active.

There are 6 different levels from None (No Severity) to Critical (Most Severe).

Out of Range is a special severity level to denote unexpected values that are outside the normal range of values.

Severity Levels

The table below shows the Severity level and the decimal value that will be written to the event's corresponding EventX_Status register, where X is the index of the event.
Severity Status Value
None 0
Warning 257
Minor 513
Major 769
Critical 1025
Out of Range 65281
