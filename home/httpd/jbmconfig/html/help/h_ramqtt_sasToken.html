Description

Multiple SAS Token formats are supported. The SAS Token can be copied from the Device Explorer stand alone application or using the IoT Hub Explorer CLI tool. The Azure Shared Access Signature (SAS) Token should contain the following information:

SharedAccessSignature sr=<IoT_Hub_Name>.azure-devices.net%2Fdevices%2F<Device_Name>&sig=<Base_64_Encoded_Signature>&se=<Expiry_Time_in_Seconds_from_Epoch>
