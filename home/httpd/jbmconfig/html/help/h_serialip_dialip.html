Description

This option appears only, if the Connect Mode is set to AT Command and then if the Dial IP option is set to Yes, the IndustrialPro router expects to receive AT commands to initialize the serial port, and also expects to receive ATDT in one of the following formats as a destination IP/Port address for connectivity to the host processing server.

ATDT192.168.0.1,5000,1

ATDT192.168.0.1[5000]

ATDT1921680000015000

ATDT192168000001/5000

ATDTdns.com,5000

Recommended Setting

No
