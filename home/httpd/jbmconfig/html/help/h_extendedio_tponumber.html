Description

TPO period:
Time Proportioned Outputs (TPO) are outputs that turn on and off in proportion to an analog value. Typically, the output will turn on and off once during the specified TPO period (cycle time). Specify a cycle time for all enabled TPO outputs in the module. The range for the cycle time is 20ms to 10 minutes (600,000ms). Each TPO output will pulse ON and OFF once during each cycle period, unless a Minimum OFF/ON time is specified.
The default TPO period value is 1000ms.
