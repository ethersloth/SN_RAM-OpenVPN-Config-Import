Description

The Master API Key is used to automatically register a device to the cloud.

After device registration is complete, the Device API Key and Device ID will be added to the RAMQTT configuration and the Master API Key will be removed.

Auto Registration Across Mulitple Devices

The Master API Key can be reused to allow auto registration across multiple devices. Once the desired configuration is finished, instead of clicking Apply, click the Save button. The configuration will be saved and the RAMQTT service will not be started. This avoids RAMQTT starting the registration process and removing the Master API Key from the configuration. Navigate to Configuration Manager page and export the RAMQTT subsystem. Once the subsystem is exported, this configuration can be reused to auto register other devices.
