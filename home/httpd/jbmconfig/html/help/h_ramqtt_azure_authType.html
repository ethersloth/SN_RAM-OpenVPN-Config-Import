Description

The Authentication Type determines how RAMQTT will authenticate with the Azure IoT Hub.

SAS Token
RAMQTT will used an already existing/generated SAS Token to authenticate with the Azure IoT Hub.

Device Connection String
RAMQTT will use a Device Connection String to generate a SAS Token to authenticate with the Azure IoT Hub. Each time the SAS Token expires, RAMQTT will use the Device Connection String to generate a new SAS Token.

Self Signed Certificates
RAMQTT will use the selected Device Certificate file, Device Certificate Key file, and a Root Certificate Authority file to authenticate the Broker.
